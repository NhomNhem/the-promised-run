## Enemy FSM (đã có sẵn)

Code đã có sẵn trong project:

- `EnemyController.cs` - Main controller với StateMachine
- `EnemyBaseState.cs` - Base class cho các states
- `EnemyIdleState.cs` - Disguised/Idle state
- `EnemyChaseState.cs` - Chase player
- `EnemyAttackState.cs` - Attack với cooldown
- `EnemyDeathState.cs` - Death animation + destroy

Transitions:
- Idle → Chase: khi player vào aggro range
- Chase → Idle: khi player ra khỏi aggro range
- Chase → Attack: khi player trong attack range
- Attack → Chase: khi player ra khỏi attack range
- Any → Death: khi health <= 0
